pySweeper

pySweeper is a recreation of the popular game Minesweeper using the Python programming language. It uses the Tkinter library as well as the Python Imaging Library for its interface enabling for a faithful recreation of the beloved classic.

Ace of Space

Ace of Space is a space shooter game inspired by the game Asteroids. It features a local 1v1 play between two players, with three different and unique fighters to choose from, each with its own strengths and weaknesses. It was developed using the Java programming language, specifically the JavaFX platform.
